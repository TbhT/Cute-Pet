# CutePet

> This project is a study case for Framework7-vue. Online address is [https://hwnmfv.coding-pages.com/](https://hwnmfv.coding-pages.com/).

## features

- feeds list
- like, comment
- multi tabs check
- swiper check
- form upload images


## Install

`npm install`

## debug or dev

`npm run build-dev`

## build prod

`npm run build-prod`
